# AirBnB Clone

This project is Command-Line Interpreter for AirBnB Clone.

## Usage

To start up the interactive Mode Run this command
```c
$ ./console.py
```

## Available Commands

```
+----------+---------------------------------------------------------------+---------------------------+
|Command   |  Syntax                                                       | Description               |
+----------+---------------------------------------------------------------+---------------------------+
|create    | create <ClassName>                                            | create an object          |
+----------+---------------------------------------------------------------+---------------------------+
|show      | show <ClassName> <id> OR                                      | shows object of class     |
|          | <ClassName>.show(<id>)                                        |                           |
+----------+---------------------------------------------------------------+---------------------------+
|destroy   | destroy <ClassName> <id> OR                                   | destroys object of class  |                 
|          | <ClassName>.destroy(<id>)                                     |                           |                          
+----------+---------------------------------------------------------------+---------------------------+
|update    | update <ClassName> <id> <attribute name> <attribute value>    | updates object attributes |
|          | <class name>.update(<id>, <attribute name>, <attribute value>)|                           |
+----------+---------------------------------------------------------------+---------------------------+
|all       | all OR all <ClassName>                                        | prints all objects        |
+----------+---------------------------------------------------------------+---------------------------+
|help      | help <command>                                                | to get help about commands|                    
+----------+---------------------------------------------------------------+---------------------------+
```
